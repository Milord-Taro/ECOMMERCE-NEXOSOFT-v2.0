# Nexosoft Ecommerce (.NET)

Aplicación web desarrollada en ASP.NET Core MVC para la gestión integral de tiendas ferreteras, orientada a pequeñas y medianas empresas en Colombia.

## Tecnologías usadas

- C#
- ASP.NET Core MVC
- Entity Framework
- MySQL
- HTML, CSS, Bootstrap
- JavaScript

## Funcionalidades principales

- Gestión de productos
- Carrito de compras
- Gestión de pedidos
- Control de inventario
- Sistema de autenticación y roles (admin, cliente, vendedor)
- Validación de formularios
- Gestión de usuarios

## Arquitectura

- Patrón MVC
- Separación de capas
- Aplicación de principios SOLID

## Base de datos

- Diseño relacional en MySQL
- Uso de consultas SQL (joins, relaciones)

## Características destacadas

- Sistema de validaciones centralizado
- Control de acceso por roles
- Módulo de inventario con trazabilidad de movimientos

## Documentación

Incluye:
- Casos de uso
- Diagramas UML
- Manual de usuario

## Cómo ejecutar el proyecto

1. Clonar repositorio
2. Configurar conexión a MySQL
3. Ejecutar migraciones (si aplica)
4. Ejecutar proyecto en Visual Studio 2022 (IIS Express)
